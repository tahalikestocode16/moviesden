# MoviesDen

A modern movie website featuring information about movies, events, galleries, and more.

## Features

- Browse movie collections
- View upcoming releases
- Explore IMAX experiences
- Event information
- Photo gallery
- Awards and FAQ section
- Contact information

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A local web server (optional, for development)

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser to view the site

### Hosting

To host this website:

1. Upload all files to your web hosting service
2. Ensure the folder structure is maintained:
   - HTML files in the root directory
   - Images in the `images` folder
   - Font files (`.otf`, `.ttf`) in the root directory

3. Set `index.html` as your default/home page
